−5%−1%0+1%+5%

hotokei とは

大きな時計と日経平均を1画面に集約したシンプルなダッシュボードです。サブディスプレイや常時表示タブにそのまま置いておき、視界の隅で 「時刻」と「相場の温度」 を同時に掴めます。

⚠ 重要な注意事項（はじめに必ずお読みください）

表示している日経平均は Yahoo Finance 経由で取得しており、1〜数十秒の遅延があります。

絶対にトレード判断には使わないでください。 取引判断には必ず証券会社（SBI、楽天、auカブコム等）のリアルタイム表示を使用してください。

特に 引け前後（15:25〜15:35） は SBI 等のリアルタイム配信と数百円ズレることがあります。

主な機能

大きな時計

時 / 分 / 秒のうち1つを大きく強調表示。タップで強調する桁が 時 → 分 → 秒 → 分 の順に切り替わります。

日経平均（リアルタイム近く）

東京市場 9:00〜15:30 の間、2秒間隔で日経平均（^N225）を取得し、現在値・前日比・騰落率を3行で表示。市場時間外は前回値（多くの場合 当日終値）を固定表示します。

背景グラデーションで「相場の温度」

前日比%に応じて背景が 連続的に 変化します。

プラス側 → 0% は暗色、+5%以上で赤最大（赤が濃くなるほど強い上昇）

マイナス側 → 0% は暗色、−5%以下で青最大（青が濃くなるほど強い下落）

音声通知（±1%バンド毎）

整数1%のバンド（+1, +2, …, +5, −1, −2, …, −5）を跨ぐ瞬間に音声で読み上げ。例: 0.8% → 1.2% で 日経 プラス1パーセント。

設定 → 「日経 音声通知」で ON/OFF（デフォルト OFF）。離席中の大きな変動察知に便利。

引け処理中 / 確定中の表示

引け前後の値が信用しにくい時間帯にオレンジで点滅警告。

15:29:00 〜 15:30:29 → 引け処理中

15:30:30 〜 15:34:59 → 確定中

寄り付き10秒前カウントダウン

9:00 寄り付きの10秒前から「開始まで X秒」表示 + ティック音で寄り意識を喚起。

市場開閉ドット

画面下「TOKYO / JST」左の点。緑 = 開場中、グレー = 閉場中。営業日（祝日除外）のみ点灯。

定期点滅（オプション）

市場時間中、設定間隔（1/3/5分）ごとに背景が一瞬白く反転。時間経過の感覚を保てます。

操作方法

  • 画面タップ — 強調する桁を切り替え
  • 左スワイプ — 設定パネルを開く
  • 右スワイプ — 設定パネルを閉じる
  • 右下の ⚙ アイコン — 設定パネルを開く
  • Esc キー — 設定パネルを閉じる

設定項目

  • 点滅 — 市場時間中の背景フラッシュ（ON/OFF）
  • 通知音 — 点滅／カウントダウンの音（ピピッ／無音）
  • 間隔 — 点滅と音の間隔（1分／3分／5分）
  • 日経 音声通知 — ±1%跨ぎ時の読み上げ（ON/OFF）

設定は localStorage に保存され、次回アクセス時も保持されます。

こんな使い方がおすすめ

  • サブディスプレイの常時表示で「相場の温度」を背景色で感じる
  • 離席中に音声通知で大きな変動を察知
  • ザラ場中の時間意識を時計と一緒に保つ
  • 朝 8:59 から 9:00 寄り付きをカウントダウンで体験
  • 引け間際の判断タイミングを「引け処理中」表示で把握
