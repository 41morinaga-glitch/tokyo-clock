hotokei

東京市場のための時計 — サブディスプレイの片隅に。

hotokei とは

東京証券取引所のザラ場時間(9:00〜15:30)を体感するための、ミニマルな時計です。サブディスプレイや常時表示タブにそのまま置いておくことで、視界の隅で 「今、市場のどこにいるか」 を感じ取れます。

寄り付きまでのカウントダウン、引け前後の状態表示、市場開閉の可視化など、東京市場の時間構造そのものを表現することに特化しています。

主な機能

大きな時計

時 / 分 / 秒のうち1つを大きく強調表示。タップで強調する桁が 時 → 分 → 秒 → 分 の順に切り替わります。

寄り付き10秒前カウントダウン

9:00 寄り付きの10秒前から「開始まで X秒」表示 + ティック音で寄り意識を喚起します。

引け処理中 / 確定中の表示

引け前後の状態をオレンジで点滅表示。

15:29:00 〜 15:30:29 → 引け処理中

15:30:30 〜 15:34:59 → 確定中

市場開閉ドット

画面下「TOKYO / JST」左の点。緑 = 開場中、グレー = 閉場中。営業日（祝日除外）のみ点灯。

定期点滅（オプション）

市場時間中、設定間隔（1/3/5/10/15/30分）ごとに背景が一瞬白く反転。時間経過の感覚を保てます。

操作方法

  • 画面タップ — 強調する桁を切り替え
  • 左スワイプ — 設定パネルを開く
  • 右スワイプ — 設定パネルを閉じる
  • 右下の ⚙ アイコン — 設定パネルを開く
  • Esc キー — 設定パネルを閉じる

設定項目

  • 点滅 — 市場時間中の背景フラッシュ（ON/OFF）
  • 通知音 — 点滅／カウントダウンの音（ピピッ／無音）
  • 間隔 — 点滅と音の間隔（1分／3分／5分／10分／15分／30分）

設定は localStorage に保存され、次回アクセス時も保持されます。

こんな使い方がおすすめ

  • サブディスプレイの常時表示 — 作業の邪魔にならず、視界の隅で市場時間を意識
  • ザラ場中の時間感覚を保つ — 大きな時計で時間の流れを把握
  • 朝 8:59 から 9:00 寄り付きをカウントダウンで体験
  • 引け間際のタイミングを「引け処理中」表示で把握
